# 图片 / PDF / 海报 / 腾讯问卷 详细要求

---

## 1. PDF 报告

标题示例:`果汁哥世界杯淘汰赛第一轮分析报告`

要求:
- 每页内容充实,不写空泛方法论,不堆大段文字;
- 用表格、卡片、分层、结论框;加入果汁哥元素;
- 明确"90分钟常规时间"口径;保留平局风险;
- 页码正确;中文无错别字;标题编号不重复;避免半页空白;
- 复杂中文图表尽量用程序排版,不要只靠 AI 画图。

推荐页结构(以 16 场淘汰赛为例):
1. 封面:标题、口径、核心摘要
2. 核心结论总览
3. 16场胜平负总表(上半区)
4. 16场胜平负总表(下半区)
5. 逐场卡片 1–4
6. 逐场卡片 5–8
7. 逐场卡片 9–12
8. 逐场卡片 13–16
9. 状态/轮休/伤停/停赛/气候修正
10. 最终判断页

---

## 2. 海报(群发图)

- 竖版;标题醒目;蓝白底 + 果汁色点缀;含果汁哥头像/元素;内容分块;
- **不要**出现"拿得准/拿不准"这类词;改用"重点方向 / 相对看好 / 分歧较大 / 需防平局";
- 底部加一句话总结;文字不重叠;不要大量小字;中文必须清楚。

---

## 3. 腾讯问卷(统计群友胜平负预测)

要求:
- 第 1 题:群昵称(填空);
- 每场一个单选题,每题必答;只统计 90 分钟常规时间;
- 选项为:主队胜 / 平 / 客队胜;
- 末尾可加开放题:最可能爆冷或打平的比赛;
- 默认不设比分题,除非用户明确要求。

生成问卷的提示词模板:

```text
请帮我创建一份腾讯问卷,用于统计群友对世界杯淘汰赛第一轮比赛的常规时间胜平负预测。

问卷标题:
果汁哥世界杯淘汰赛第一轮胜平负预测

问卷说明:
本问卷用于统计群友对世界杯淘汰赛第一轮比赛的赛前判断。所有题目均只统计90分钟常规时间结果,不包含加时赛和点球大战。如果90分钟战平,即选择"平"。本问卷仅用于群内赛事讨论和内容统计,不代表最终赛果。

题型要求:
1. 第1题为填空题:请输入你的群昵称。
2. 第2题开始为单选题,每场比赛设置一道题。
3. 每道比赛题必须设置为必答。
4. 每道题选项为:主队胜 / 平 / 客队胜。
5. 不需要设置比分题。

请按以下赛程生成题目:
{赛程列表}

最后增加一道可选填空题:
你认为本轮最可能爆冷或打平的比赛是哪一场?为什么?
```
