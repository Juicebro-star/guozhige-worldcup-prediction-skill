## 2. 单场样例:强方向 + 防守剧本

```text
这场我不绕,直接给:

90分钟主判断:{强队}胜。
主比分:{强队} 2-0 {弱队}。
备选比分:{强队} 2-1 {弱队}。
防守剧本:1-1。

{弱队}不是会明显崩盘,但{强队}赢面明显更高。

原因:
1. {弱队}中前场{伤缺/短板},削弱推进、反击和边路爆点。
2. {强队}前场个人能力和阵容完整度更好。
3. {天气/赛地变量}不利于{弱队}长时间高强度跑动,{强队}可以靠控球降速处理。

最终结论:
{强队} 2-0 {弱队}。
如果{弱队}前60分钟守住,则防1-1。
```

## 3. 单场样例:强方向但不看大胜

```text
这场我明确给:

90分钟主判断:{强队}胜。
主比分:{强队} 1-0 {对手}。
备选比分:{强队} 2-0 {对手}。
防守剧本:0-0 / 1-1。

{强队}方向是胜,但我不看大胜。

原因:
1. {强队}纸面实力更强,但{近期状态短板}。
2. {对手}能守、能拖、能把比赛变难。
3. 淘汰赛{阶段}{强队}会更重视控制风险,不一定从一开始就大开大合。

最终结论:
{强队} 1-0 {对手}。
如果{强队}前60分钟不进球,就要防平。
```

## 4. 单场样例:防平结构

```text
{硬骨头}这场我明确给:

90分钟主判断:平局。
主比分:{热门队} 1-1 {硬骨头}。
备选:{热门队} 2-1 {硬骨头}。

我的倾向不是{硬骨头}胜,而是:{硬骨头}有能力把{热门队}拖平,但90分钟直接赢,难度还是偏大。

原因:
1. {硬骨头}不是普通弱队,{小组赛表现}说明有强度和纪律。
2. {热门队}更有传统强队底子,但没有绝对碾压优势。
3. 这场{情绪/对抗/节奏}会很重,不会简单。

最终结论:
常规时间:{热门队} 1-1 {硬骨头}。
如果90分钟分胜负,我更偏{热门队} 2-1。
```
