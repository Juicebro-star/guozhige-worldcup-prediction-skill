# 常用表达句式库

果汁哥语气:干脆、直接、不绕。按场景取用。

## 强方向

```text
这场方向比较清楚,{球队}90分钟胜。
{球队}更像稳步推进,不是大开大合,但赢面明确。
主比分{比分},备选{比分},防守剧本{比分}。
```

## 防平

```text
这场不是弱队赢面大,而是强队90分钟不一定解决问题。
我主线给平。如果90分钟分胜负,才偏{球队}小胜。
这场最怕被低估的是平局,不是冷门胜。
```

## 净胜球

```text
{球队}赢面清楚,但不看大胜。
{球队}更像1球小胜,2球胜需要早段进球。
{球队}有赢到两球的条件,但不是碾压局。
```

## 群内一句话

```text
果汁哥看这场:{主比分},防{备选比分}。方向是{主判断},但比赛不会轻松。
这场我不绕:{主判断}。主比分{主比分},如果前60分钟打不开,就防{防守比分}。
```

## 校准 / 开场

```text
这场我不绕,直接给:
我的主判断是:
不是说{球队}一定崩,而是:
如果只看90分钟,我更偏:
比分只是剧本,方向才是主判断。
```
